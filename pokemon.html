Pokemon 'Emulator'

Company: Epicodus Group Project

Project: Javascript-based web-browser video game inspired by the original Pokemon for Gameboy.

Role:Javascript Development

Tech:Vanilla JavaScript

One month into code school, having just been introduced to the basics of JavaScript (and I mean the BASICS), I was tasked with pair-programming Javascript methods that controlled the behavior of Pokemon characters, duking it out in a browser-based homage to the original Pokemon Gameboy game.

View the live demo here. After hitting the power button, it takes 30 seconds to load, just like the original, so be patient!
