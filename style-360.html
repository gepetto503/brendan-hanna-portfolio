Style 360

  • Role: Front End Development
  • Tech: React JS

As an intern at SightWorks Digital Products, I was tasked with turning design mockups for a one page scrolling website into a UI built out of responsive React JS components. In the process, I was also to create a style guide for future designers and developers

This process sparked one of the questions I'm most interested in today: how can designers and developers collaborate more effectively -- not only by using design systems, but by applying empathy to collaboration?
