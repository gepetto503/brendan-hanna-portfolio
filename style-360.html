Style 360

Company: SightWorks

Project: Responsive one-page web site for hair stylists

Role: Front-End Web Development

Tech: React JS, SASS, create-react-app (NPM package)

SightWorks had an idea for a new product called Style360 and needed the UI prototyped in React. They had not used React in production before and asked my co-intern and I to a) learn React, b) decide on a development environment and any front-end frameworks and c) build the UI.

I took the lead in learning React and brought useful information to the team which directly influenced the direction of the project.
